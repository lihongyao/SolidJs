# 概述

## i18n 常见需求

- **文案翻译**（按钮、标题、提示语）

-  **动态切换语言**（比如中文 ↔ 英文）

-  **默认语言策略**（浏览器语言 / 用户配置 / 系统默认）

-  **内容维护**（方便开发、翻译和后续扩展）

## 开发维护的思路

### 📁 文件组织

最常见做法是**多语言资源文件**，比如：

```
/locales
  ├── en.json
  ├── zh.json
```

# 技术实现流程

1 安装依赖

```shell
$ pnpm add i18next i18next-browser-languagedetector i18next-resources-to-backend
```

- `i18next`：核心翻译引擎

- `i18next-browser-languagedetector`：检测浏览器语言

- `i18next-resources-to-backend`：动态加载 JSON 资源
